# RemoteControl
基于TCP的远程桌面控制软件（南京邮电大学软件设计题目）

## 项目简介
开发一款类似向日葵的远程桌面控制系统。被控端捕获屏幕图像并编码为JPEG后通过TCP发送至控制端，控制端实时显示画面并将本地鼠标、键盘事件转发至被控端模拟执行，实现局域网内远程操作。

## 技术栈
C++；Winsock2（TCP协议），自定义数据包结构（FDataHeader）；GDI+ 屏幕捕获与JPEG编码，HBITMAP位图操作；Windows API（SetCursorPos、SendInput）；多线程（接收线程、发送线程、输入处理线程）。

## 完成效果
在1920×1080分辨率下，屏幕捕获+JPEG编码每帧约15-20 ms，图像大小15-300KB，局域网内帧率可达20-30fps；鼠标、键盘事件响应延迟30-60 ms，操作流畅无卡顿；程序支持高DPI自适应，连接异常时可安全退出，经长时间运行无内存泄漏。成绩评定为“优秀”。
